Guide for Newbie

Follow these steps from zero until the texture pack is successfully installed.

Important: Make sure you have the Steam version of Geometry Dash (not cracked/pirated). Geode is recommended with the official version. Versions 2.200+ are recommended too.
1

Download Geode Installer

Visit the official Geode page and download the latest installer according to your OS.

geode-sdk.org/install
Geode supports Windows, Android, macOS, iOS, and Linux.
2

Run Installer

Open the installer file. The installer will automatically detect the location of Geometry Dash on your system. Click "Install" and wait for the process to complete. Do not close the window until a confirmation appears.

3

Open Geometry Dash

Launch GD via Steam. If successful, you will see the Geode splash screen (diamond logo) during loading, and the Geode icon will appear in the top right corner of the main menu.

Geode logo appears? Installation successful! Proceed to the "Install Pack" tab.
4

Verify Geode Folder (Optional)

Click the Geode icon in the menu → Geode Settings to view info. The Geode folder is located at:

%localappdata%/GeometryDash/geode/
/sdcard/GeometryDash/geode/
Make sure Geode is installed and running. See the tab "Install Geode" if not.

1

Choose a Texture Pack

Scroll to the "Texture Packs" section on this page, or use the search bar to find the pack you want. Click the "Download" button on the pack card.

2

Choose the Correct Resolution

Each pack is available in two resolution options. Choose the one that suits your platform:

PC Resolution

Texture with high resolution for PC/Laptop. Maximum detail quality, larger file size. Use this if playing on a computer.

Mobile Resolution

Texture with low resolution for Android. Stable performance, smaller file size. Must be used if playing on a smartphone/tablet.

3

Install via Geode (Recomended)

Open Geometry Dash → click the Geode icon → select "Mods" → click "Open Folder" → navigate to the mods folder → place the pack file/folder here → restart the game.

%localappdata%/GeometryDash/geode/mods/
4

Manual installation (alternative)

If the method above doesn't work, extract the downloaded zip file, then copy the entire pack folder to the following location:

%localappdata%/GeometryDash/geode/mods/[nama-pack]/

Make sure each pack is in its own folder and contains a pack.json file in its root.

5

Activate & Play!

Restart game → open Geode Mods menu → find your pack → toggle ON. Texture will be active immediately without needing to restart again!

Congratulations! The texture pack is now installed. You can toggle it OFF/ON at any time to switch packs.
Common issues and their solutions. If your issue isn’t listed here, visit Discord Geode for help.
Geode isn't showing up / isn't loading
  • Make sure your Geode version matches your GD version (check in Geode Settings)
  • Make sure you are using the official Steam version of the game (not a pirated version)
  • Re-install Geode from the official website
  • Disable your antivirus temporarily while installing (it may cause false positives)
  • Delete the old geode/ folder and reinstall from scratch
  • Check geode/logs/ for error messages
The texture pack isn't showing up in the mods list
  • Make sure the file is in the geode/mods/ folder (not in a subfolder)
  • Ensure there is a pack.json file in the root folder of the pack
  • Restart the game after placing the file
  • Check the crash log in geode/logs/
  • Make sure the pack is compatible with your Geode version
The game crashes after installing the texture pack
  • The pack might not be compatible with your Geode/GD version
  • Delete the newly installed pack and check if the game runs normally
  • Don't enable two packs that modify the same elements simultaneously
  • Re-download the pack (the file might be corrupted during download)
  • Check geode/crashlogs/ for detailed error information
The texture is broken / not displaying correctly in the game
  • Download the correct version (PC vs Mobile) — don't get them mixed up
  • Don't mix textures from different packs for the same elements
  • Turn off all other packs and enable them one by one for testing
  • Make sure there are no leftover files from old packs in the Resources folder
Geode Android isn't installing
  • GD Android must be version 2.205+ from the Play Store
  • Download the Android installer from the official page
  • Allow storage access when prompted by the installer
  • Install from Unknown Sources if necessary (Settings → Security)
  • Some devices with custom ROMs may have compatibility issues
Is it safe to play online with a texture pack?

100% safe. Texture packs are client-side only. Other players won't see any changes, and you won't be banned because texture packs don't affect gameplay, scores, or input.

One Pack at a Time

Activate only one texture pack at a time. Having multiple packs enabled simultaneously can cause conflicts and make the display messy.

Choose the Right Resolution

PC for computer, Mobile for Android. PC packs on Android might cause lag due to textures being too large for mobile GPUs.

Backup Before Changing

Before installing a new pack, backup the mods folder so you can revert to the old pack at any time.

Update Geode Regularly

Always update Geode to the latest version when there are major GD updates. Older versions often aren't compatible.

Only Trusted Sources

Download packs only from trusted sources like this page or the Geode hub. Packs from unknown sources might be risky.

Mandatory Folder Structure

Each pack must have a mod.json file in its root folder. Without this file, Geode won't recognize the pack at all.

Performance & FPS

Texture packs generally don't significantly impact FPS since they only replace static images (sprites). However, high-resolution packs on low-end PCs might slightly reduce performance. If FPS drops drastically, try using the Mobile version or disable the pack.
