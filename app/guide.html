Guide untuk Pemula

Ikuti langkah-langkah berikut dari nol sampai texture pack berhasil terpasang

Penting: Pastikan kamu punya Geometry Dash versi Steam (bukan crack/pirated). Geode hanya kompatibel dengan versi resmi. Versi 2.200+ direkomendasikan.
1

Download Geode Installer

Kunjungi halaman resmi Geode dan download installer terbaru sesuai OS kamu.

geode-sdk.org/install
Geode mendukung Windows dan Android. macOS/iOS belum tersedia secara resmi.
2

Jalankan Installer

Buka file installer. Installer akan otomatis mendeteksi lokasi Geometry Dash kamu. Klik "Install" dan tunggu proses selesai. Jangan tutup window sampai muncul konfirmasi.

3

Buka Geometry Dash

Jalankan GD via Steam. Jika berhasil, kamu akan melihat splash screen Geode (logo berlian) saat loading, dan ikon Geode muncul di pojok kanan atas menu utama.

Logo Geode muncul? Instalasi berhasil! Lanjut ke tab "Install Pack".
4

Verifikasi Folder Geode (Opsional)

Klik ikon Geode di menu → Geode Settings untuk melihat info. Folder Geode berada di:

%localappdata%/GeometryDash/geode/
/sdcard/GeometryDash/geode/
Pastikan Geode sudah ter-install dan berjalan. Lihat tab "Install Geode" jika belum.
1

Pilih Texture Pack

Scroll ke bagian "Texture Packs" di halaman ini, atau gunakan search bar untuk menemukan pack yang kamu inginkan. Klik tombol "Download" pada card pack.

2

Pilih Resolusi yang Tepat

Setiap pack tersedia dalam dua varian resolusi. Pilih sesuai platform kamu:

PC Resolution

Texture beresolusi tinggi untuk PC/Laptop. Kualitas detail maksimal, ukuran file lebih besar. Gunakan ini jika bermain di komputer.

Mobile Resolution

Texture beresolusi rendah untuk Android. Performa stabil, ukuran file kecil. Wajib digunakan jika bermain di HP/tablet.

3

Install via Geode (Direkomendasikan)

Buka Geometry Dash → klik ikon Geode → pilih "Mods" → klik "Open Folder" → masuk folder mods → taruh file/folder pack di sini → restart game.

%localappdata%/GeometryDash/geode/mods/
4

Install Manual (Alternatif)

Jika metode di atas tidak bisa, ekstrak file zip yang didownload, lalu copy seluruh folder pack ke lokasi berikut:

%localappdata%/GeometryDash/geode/mods/[nama-pack]/

Pastikan setiap pack berada di folder tersendiri dan mengandung file mod.json di root-nya.

5

Aktifkan & Mainkan!

Restart game → buka menu Geode Mods → temukan pack kamu → toggle ON. Texture langsung aktif tanpa perlu restart lagi!

Selamat! Texture pack sudah terpasang. Kamu bisa toggle OFF/ON kapan saja untuk berganti pack.
Masalah paling umum dan solusinya. Jika masalahmu tidak ada di sini, kunjungi Discord Geode untuk bantuan.
Geode tidak muncul / tidak loading
  • Pastikan versi Geode cocok dengan versi GD kamu (cek di Geode Settings)
  • Pastikan game versi Steam resmi (bukan pirated)
  • Re-install Geode dari geode-sdk.org/install
  • Disable antivirus sementara saat install (sering false positive)
  • Hapus folder geode/ lama, lalu install ulang dari awal
  • Cek geode/logs/ untuk error message
Texture pack tidak muncul di daftar mods
  • Pastikan file berada di geode/mods/ (bukan subfolder yang salah)
  • Pastikan ada file mod.json di root folder pack
  • Restart game setelah menaruh file
  • Cek crash log di geode/logs/
  • Pastikan pack kompatibel dengan versi Geode kamu
Game crash setelah install texture pack
  • Pack mungkin tidak kompatibel dengan versi Geode/GD kamu
  • Hapus pack yang baru di-install, cek apakah game normal
  • Jangan aktifkan dua pack yang mengubah texture elemen sama secara bersamaan
  • Download ulang pack (file mungkin corrupt saat download)
  • Cek geode/crashlogs/ untuk detail error
Texture berantakan / tidak sesuai di game
  • Download versi yang benar (PC vs Mobile) — jangan tertukar
  • Jangan campur texture dari pack berbeda untuk elemen yang sama
  • Matikan semua pack lain, aktifkan satu per satu untuk tes
  • Pastikan tidak ada leftover file dari pack lama di folder Resources
Geode Android tidak ter-install
  • GD Android harus versi 2.205+ dari Play Store
  • Download installer Android dari halaman resmi
  • Izinkan akses storage saat diminta oleh installer
  • Pasang dari Unknown Sources jika perlu (Settings → Security)
  • Beberapa device dengan custom ROM mungkin memiliki masalah kompatibilitas
Bermain online dengan texture pack — aman?

Aman 100%. Texture pack hanya bersifat client-side. Player lain tidak melihat perubahan apapun, dan kamu tidak akan di-ban karena texture pack tidak mempengaruhi gameplay, score, atau input.

Satu Pack Sekali

Aktifkan hanya satu texture pack pada satu waktu. Beberapa pack sekaligus bisa konflik dan membuat tampilan berantakan.

Pilih Resolusi Tepat

PC untuk komputer, Mobile untuk Android. Pack PC di Android bisa lag karena texture terlalu besar untuk GPU mobile.

Backup Sebelum Ganti

Sebelum install pack baru, backup folder mods supaya bisa kembali ke pack lama kapan saja.

Update Geode Secara Rutin

Selalu update Geode ke versi terbaru saat ada update GD besar. Versi lama sering tidak kompatibel.

Sumber Terpercaya Saja

Download pack hanya dari sumber terpercaya seperti halaman ini atau Geode hub. Pack dari sumber asing bisa berisiko.

Struktur Folder Wajib

Setiap pack harus punya mod.json di root folder. Tanpa file ini, Geode tidak akan mengenali pack sama sekali.

Performa & FPS

Texture pack umumnya tidak mempengaruhi FPS secara signifikan karena hanya mengganti gambar statis (sprite). Namun pack dengan resolusi sangat tinggi di PC low-end bisa sedikit menurunkan performa. Kalau FPS turun drastis, coba gunakan versi Mobile atau nonaktifkan pack.
